Distributed NoSQL Key-Value Database

● I created a Key-Value database that is sharded and replicated.
● The db has no single point of failure because it uses Paxos to replicate data across nodes.
● Not all nodes are responsible for all data; groups of nodes save certain shards of the data.
● Shard reorganization code allows the db to scale horizontally and add more nodes.

  • Date: December 2015

Connect Me

I worked on a flask web app that predicts future salaries of your LinkedIn connections and gives you recommendations on whom you should ask for job referrals. This project won 3rd place at a UC Berkeley hackathon that was hosted by Intuit.

I designed the strategy on applying linear regression models and cross validation to do salary predictions. I also implemented the machine learning code using Python and the Scikit-learn module.

  • Date: September 2016
  • Link to Project: https://github.com/andrewyli/hackintoit2016

Vaccine Usage Predictor

This project involved building linear regression models with Python's Scikit-learn library to predict vaccination usage in Denmark. Cross validation was performed in order to reduce model over-fitting. The MMR vaccination predictor was accurate enough to achieve a root-mean-square-error that was less than 15% of the data range.

  • Date: April 2016
